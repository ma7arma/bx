# Bixah QR Shock Demo — Mobile First
Mobile‑optimized, static, no‑backend demo that shows device fingerprinting and (consent) camera/location.

## Publish on GitHub Pages
1. Open `ma7arma/bx`.
2. Upload these files to the repo root (replace existing if needed).
3. Settings → Pages → Deploy from branch → `main` → `/root`.
4. Open: `https://ma7arma.github.io/bx/?mode=shock`

## Notes
- Single‑column layout, larger tap targets, collapsible sections, bottom‑sheet intro.
- No network calls, no storage. Camera/Location only after explicit consent.
